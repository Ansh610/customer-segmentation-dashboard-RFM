# 📊 Customer Segmentation Dashboard (RFM + K-Means)

## 🚀 Overview

This project performs **Customer Segmentation using RFM Analysis (Recency, Frequency, Monetary)** and **K-Means Clustering**, and presents insights through an **interactive Streamlit dashboard**.

The goal is to help businesses understand customer behavior and design **targeted marketing strategies**.

---

## 🎯 Objectives

* Segment customers based on purchasing behavior
* Identify high-value and at-risk customers
* Provide actionable business insights
* Build an interactive analytics dashboard

---

## 🧠 Techniques Used

* RFM Analysis (Recency, Frequency, Monetary)
* K-Means Clustering (Machine Learning)
* Data Cleaning & Feature Engineering
* Data Visualization
* Dashboard Development (Streamlit)

---

## 🛠️ Tech Stack

* **Python**
* **Pandas, NumPy**
* **Scikit-learn**
* **Matplotlib, Seaborn**
* **Plotly**
* **Streamlit**
